== Description ==

Fullscreen Menu is a free plugin to create custom fullscreen menu quickly in your wordpress website (user-friendly, highly customizable, responsive)


Documentation & Demo : http://najih.net/fullscreen-menu-wordpress-plugin/

== Installation ==


1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to Appearance > Menus and create a menu and assign it to 'fullscreen Menu Plugin'
4. go to 'fullscreen Menu Plugin' option to customize it
